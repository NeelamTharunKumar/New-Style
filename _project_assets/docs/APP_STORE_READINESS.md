# App Store Readiness Checklist

## Product

- [ ] Final app name confirmed
- [ ] App icon created
- [ ] Splash screen created
- [ ] Screenshots for Android generated
- [ ] Screenshots for iPhone generated
- [ ] Screenshots for iPad if supported
- [ ] Demo account/test instructions prepared

## Privacy/legal

- [ ] Privacy policy reviewed by legal
- [ ] Terms of service reviewed by legal
- [ ] Data safety form completed for Google Play
- [ ] App Privacy Nutrition Labels completed for App Store
- [ ] Account deletion flow documented
- [ ] Support/contact email configured

## Android

- [ ] Package name finalized
- [ ] Upload keystore generated and backed up
- [ ] Play Console app created
- [ ] Internal testing track configured
- [ ] Data safety completed
- [ ] Content rating completed
- [ ] Signed AAB uploaded

## iOS

- [ ] Bundle ID finalized
- [ ] Apple Developer account configured
- [ ] App Store Connect app created
- [ ] Signing certificate/provisioning profile created
- [ ] TestFlight internal group configured
- [ ] Export compliance answered
- [ ] Privacy nutrition labels completed
- [ ] IPA uploaded to TestFlight

## Backend

- [ ] Production API deployed
- [ ] HTTPS enabled
- [ ] Firebase auth mode enabled
- [ ] Database backups configured
- [ ] Monitoring/logging configured
- [ ] Secrets stored in cloud secret manager
- [ ] Rate limiting configured
