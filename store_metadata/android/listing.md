# Google Play Listing Draft

## App name

BharatFit AI

## Short description

India-first wardrobe assistant that creates outfits from clothes you already own.

## Full description

BharatFit AI helps you decide what to wear for college, office, dates, weddings, Haldi, Sangeet, festivals and daily life using your own wardrobe.

Designed for Indian wardrobes, it supports ethnic and western outfit combinations, practical men's styling, climate-aware suggestions and budget-conscious recommendations.

Privacy-first design: wardrobe photos are intended to stay on your device. The backend uses structured clothing features like category, color, fabric and style tags to recommend outfits.

## Keywords

wardrobe, outfit planner, Indian fashion, AI stylist, closet, kurti, saree, menswear, wedding outfit, college outfit

## Data safety draft

- Photos: processed locally by default, not uploaded by default.
- Personal info: user/account ID for authentication.
- App activity: outfit preferences and wardrobe attributes.
- Data deletion: supported through backend data lifecycle endpoints.
